# TWIS HOLO FULL-BUILD HARNESS

## Mission
Build Randy a usable creative Workshop that hides complexity, preserves local ownership, and can grow through replaceable modules.

## Hard rules
1. Do not ship another decorative mockup.
2. Every visible room must perform real work or clearly label its limits.
3. Local project folders and SQLite are authoritative.
4. Browser storage is fallback/cache only.
5. AI is optional, advisory, and replaceable.
6. Cloudflare is optional remote muscle, never the only copy of project state.
7. Never require a paid API for core use.
8. Never silently trigger billable work.
9. Engines are adapters; projects and artifacts survive engine changes.
10. Preserve source, provenance, constraints, sessions, and history.
11. No AI may publish, pay, delete permanent source, approve Canon, or expose private memory without explicit approval.
12. Import existing work instead of forcing re-entry.
13. Build ordinary mode first; advanced controls remain available but out of the way.
14. Report working, partial, scaffolded, unavailable, and hardware-test-required honestly.
15. Continue build passes until remaining gaps require external authentication, external engine installation, unavailable hardware, or user-specific data not present in the package.

## Success test
On first run Randy can:
- create or open a project;
- talk to a built-in companion or configured compatible AI;
- write and version a story;
- make and export a beat;
- import/edit/export an image;
- import/preview/organize video and audio;
- launch a research expedition;
- inspect and edit project files;
- import old folders into the Workshop;
- search all saved work;
- close and reopen without reconstructing the session;
- export a portable project capsule;
- see which modules are local, optional cloud, or potentially billable.
