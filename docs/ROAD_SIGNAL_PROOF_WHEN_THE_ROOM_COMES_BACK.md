# Road-Signal Machine Proof Test — When the Room Comes Back

This proof test extends `ROAD_SIGNAL_MACHINE_HARNESS.md` with the Evolution Skills layer.

## Evolution Skills Layer

For this song, the tested combination is:

- Lead engine: Guitar-Synth Motor
- Support engine: Polyrhythm / Off-Time Tribal
- Accent engine: Resample-Abuse + Negative-Space Drop
- Core object: the room after the party, when reality comes back

## Harness Lesson

This song proves that Road-Signal Machine must not only change lyrics. It must change the musical mechanism.

The working rule is:

1. Pick the lead engine.
2. Pick support and accent engines.
3. Build the groove and sonic mutation.
4. Let the lyrics name the sounds while the sounds answer.
5. Use negative space before impact.
6. Keep controlled damage riding a strong groove.
